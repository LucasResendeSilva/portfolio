O Problema

Uma pesquisa realizada pelo instituto locomotiva revela que existe 45 milhões de pessoas desbancarizada no Brasil. Ou seja tem muitas pessoas sem acesso a serviço bancários.
Pensando nisso, nós da Dubbox Tech decidimos criar um uma carteira digital mais que um gateway de pagamento. Foi aí que nasceu o YuPay uma fintech, pensando atender os pequenos empreendedores e pessoas das classes B, C e D. Criar algo simples e completo pensando no usuário.



A Pesquisa

Todas as decições de interface foram feira atravez de pesquisa e teste com os usuários, para melhorar o a experiência e facílitar o uso. Também pensamos no logista de pequeno negocio até o grande. Por isso fizemos brainstormings com empresarios e pesquisa qualitativa entre nossos stakeholders.



EAP - ESTRUTURA ANALÍTICA DO PROJETO



Design System

Durante o precesso de criação, percebemos que tinha a necessidade ter um Design System. Ainda mais com um projeto desse tamanho, tem aplicativo para Pessoas Físicas e Pessoa jurídica, temos dashboard para equipe interna, e ainda painel web para o lojista.
